参靠教程：http://www.ruanyifeng.com/blog/2012/08/blogging_with_jekyll.html

个人随记:

1.创建项目

$ mkdir my_blog

$ cd my_blog

$ git init

创建一个没有父节点的分支gh-pages

$ git checkout --orphan gh-pages

2.创建设置文件 在项目根目录下创建文件_config.yml $ echo "baseurl:/my_blog" > _config.yml 3.创建模板文件 在项目根目录下，创建一个_layouts目录 $ mkdir _layouts 在_layouts 下新建 default.html 格式参靠教程 4.创建文章 在项目根目录下 $ mkdir _posts 文章在此目录下新建 ，文件名格式为： 年－月－日－文章标题.html 注意编码格式为utf8. 5.创建首页 在项目根目录下新建 index.html 格式参考教程。 6.发布内容。 $ git add . $ git commit -m "my_blog first post" 前往github 创建一个名为 my_blog 的库. 把本地内容推送上去。 $ git remote add origin http://github.com/jvschen/my_blog.git $ git push origin gh-pages 过一会，访问 http://jvschen.github.io/my_blog/就可以看到新建的blog文. 7.遇到的问题 fatal: unable to access 'https://github.com/jvschen/my_blog.git/': Couldn't resolve host 'githup.com' ping https://github.com 不通 修改协议 .git/config [remote "origin"] url = ssh://git@github.com/jvschen/my_blog.git fetch = +refs/heads/*:refs/remotes/origin/* ssh 不通,修改hosts文件 C:\Windows\System32\drivers\etc host xxxxxx github.com
